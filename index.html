JavaScipt একটি সিঙ্ক্রোনাস সিঙ্গেল থ্রেডেড ল্যাংগুয়েজ। অর্থাৎ JavaScipt সিঙ্গেল কল স্ট্যাক রয়েছে তাই এটি উপর থেকে নীচের ক্রম অনুসারে এবং একটি নির্দিষ্ট সময়ে একটি কমান্ড এক্সিকিউট করে। এটি তখনই পরবর্তী লাইনে যায় যখন বর্তমান লাইনটি এক্সিকিউট করা সম্পন্ন হয়। যেমন: যদি আপনি জাভাস্ক্রিপ্ট ফাইলের এক নম্বর লাইনে console.log("1") এবং লাইন নম্বর দুই-এ console.log("2")। তারপর টার্মিমাল আপনি প্রথম লাইনে 1 এবং দ্বিতীয় লাইনে 2 দেখতে পাবেন।

ইভেন্ট লুপ এমন একটি প্রসেস্কে বোঝায় যা একটি প্রোগ্রামে ইভেন্টগুলিকে মনিটর করে এবং ইভেন্টগুলোর জন্য অপেক্ষা করে এবং পরবর্তীতে সেগুলো প্রেরণ করে। জাভাস্ক্রিপ্টে, ইভেন্ট লুপগুলি অ্যাসিঙ্ক্রোনাসভাবে একাধিক টাস্ক প্রসেস করে৷ এটি কল স্ট্যাক এবং কলব্যাক Queue মনিটর করে। এটি কল স্ট্যাকের জন্য অপেক্ষা করে এবং কল স্ট্যাক কিউ থেকে FIFO (ফার্স্ট ইন ফার্স্ট আউট)অনুসারে কোডগুলি এক্সিকিউট করে। কিউতে থাকা প্রথম ফাংশনটি প্রথমে কার্যকর করা হবে। এভাবেই ধাপে ধাপে ইভেন্ট লুপ কাজ করে

Local Storage :
  • Local storage has control over all local files.
  • Local storage can be accessible from any window
  • Data is never lost until it is deleted manually using JavaScipt
Session Storage :
  • Session storage has no control over local files.
  • Session storage can be accessible only from same tab
  • Data is lost when tab is closed
